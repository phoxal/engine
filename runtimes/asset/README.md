# phoxal-runtime-asset

`phoxal-runtime-asset` is a BLUEPRINT skeleton for staged bundle asset serving.

Target scope: serve source-shaped staged assets over `runtime/asset/get`.

Primary query: robot structure, model, component configs, and declared assets
needed by tools and validators.

Not in scope: robotics semantics, runtime graph generation, or owning any
component/runtime contract.

See `codex/BLUEPRINT.md` and `codex/BLUEPRINT_CONTRACTS.md` for the target
contract.
